cause' legal battles are played with wordings and not emotions!

### Installation Guide

```bash
git clone https://github.com/itushh/termix.git
```

```bash
cd Termix
```
<br />

Now run backend

```bash
create .env in backend folder
```

```bash
cd backend 
```

```bash
npm install
```

```bash
npm run dev
```

<br />

Now run frontend (new terminal)

```bash
cd frontend
```

```bash
npm install
```

```bash
npm run dev
```

All Set!
